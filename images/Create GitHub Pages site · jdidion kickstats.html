# Kickstats Analyses of crownfunded projects. ## Kickstarter tabletop games This is a study of certain aspects of Kickstarter funding for tabletop game projects that launched between 2009 and 2014. ### Method I wrote a ruby script that scrapes the Tabletop Game pages on Kickstarter using a combination of regular expressions (to find the project IDs ('slugs') in the search results) and the `kickscraper` library, which is a wrapper around Kickstarter's (non-public) API. Since search results are limited to 200 pages (4000 items), and there are approximately 5500 games total, I broke the searches down by total pledges. Each individual search had fewer than 4000 results and was orthogonal to the other searches, so I was able to obtain nearly all the results. I store the data in a sqlite database. I wrote a python script to fetch historical currency conversion rates between USD and the (currently) eight other currencies supported by Kickstarter using Quandl. I preprocess the data in R, and then use knitr to generate this report. I use the default R plots in this report, mainly because they're automatic to me whereas I'm still learning ggplot2; I'll convert in the next version. I generate a PDF file rather than allowing Jekyll to render the markdown because GitHub does not support Jekyll plugins, and Pandoc is required for rendering the tables I use. To reproduce, you will need the following: * Scraping (kickscrape.rb) * Ruby 2+ * slop, sqlite3 and kickscraper libraries * Exchange rates (exchange_rates.py) * Python 2.7+ * requests * Quandl account (free) and auth_token * Data processing (format.R) * R 3+ * RSQLite library * Report (report.Rmd) * R 3+ * knitr, stringr, pander, RColorBrewer libraries
